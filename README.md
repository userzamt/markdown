# Markdown
**Markdown** (–ø—Ä–æ–∏–∑–Ω–æ—Å–∏—Ç—Å—è –º–∞—Ä–∫–¥–∞—É–Ω) ‚Äî –æ–±–ª–µ–≥—á—ë–Ω–Ω—ã–π —è–∑—ã–∫ —Ä–∞–∑–º–µ—Ç–∫–∏, —Å–æ–∑–¥–∞–Ω–Ω—ã–π —Å —Ü–µ–ª—å—é –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏—è —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –≤ –ø—Ä–æ—Å—Ç–æ–º —Ç–µ–∫—Å—Ç–µ, —Å –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–º —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏–µ–º –µ–≥–æ —á–∏—Ç–∞–µ–º–æ—Å—Ç–∏ —á–µ–ª–æ–≤–µ–∫–æ–º, –∏ –ø—Ä–∏–≥–æ–¥–Ω—ã–π –¥–ª—è –º–∞—à–∏–Ω–Ω–æ–≥–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –≤ —è–∑—ã–∫–∏ –¥–ª—è –ø—Ä–æ–¥–≤–∏–Ω—É—Ç—ã—Ö –ø—É–±–ª–∏–∫–∞—Ü–∏–π (HTML, Rich Text –∏ –¥—Ä—É–≥–∏—Ö).



## –î–∏–∞–ª–µ–∫—Ç—ã –∏ —Ä–µ–¥–∞–∫—Ç–æ—Ä—ã
–ë–∞–∑–æ–≤—ã–º —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–æ–º Markdown —É–∂–µ –º–∞–ª–æ –∫—Ç–æ –ø–æ–ª—å–∑—É–µ—Ç—Å—è. –ó–∞—Ç–æ —É –Ω–µ–≥–æ —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ü–∏–∏ –∏ –¥–∏–∞–ª–µ–∫—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –¥–æ–±–∞–≤–ª—è—é—Ç –≤ —è–∑—ã–∫ –Ω–æ–≤—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏: –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏–µ HTML-—Ç–µ–≥–æ–≤, —Å–æ–∑–¥–∞–Ω–∏–µ —Ç–∞–±–ª–∏—Ü –∏ —á–µ–∫–±–æ–∫—Å–æ–≤, –∑–∞—á—ë—Ä–∫–∏–≤–∞–Ω–∏–µ —Ç–µ–∫—Å—Ç–∞, —Ä–∞–∑–Ω—ã–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –ø–µ—Ä–µ–Ω–æ—Å–∞ —Å—Ç—Ä–æ–∫–∏.

–†–∞–∑–Ω—ã–µ —Ä–µ–¥–∞–∫—Ç–æ—Ä—ã Markdown –º–æ–≥—É—Ç –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å –∏–ª–∏ –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å —á–∞—Å—Ç—å –Ω–æ–≤—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π ‚Äî —É—á—Ç–∏—Ç–µ —ç—Ç–æ –ø—Ä–∏ –≤—ã–±–æ—Ä–µ –ø–ª–∞—Ç—Ñ–æ—Ä–º—ã, –Ω–∞ –∫–æ—Ç–æ—Ä–æ–π –±—É–¥–µ—Ç–µ —Ä–∞–±–æ—Ç–∞—Ç—å.

–°–∞–º—ã–π —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω—ë–Ω–Ω—ã–π Markdown ‚Äî –¥–∏–∞–ª–µ–∫—Ç [GitHub Flavored Markdown](https://github.github.com/gfm/), –æ—Å–Ω–æ–≤–∞–Ω–Ω—ã–π –Ω–∞ [—Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ü–∏–∏ CommonMark](https://commonmark.org/). 



## –ó–∞–≥–æ–ª–æ–≤–∫–∏
–í —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–µ Markdown –µ—Å—Ç—å —à–µ—Å—Ç—å —É—Ä–æ–≤–Ω–µ–π –∑–∞–≥–æ–ª–æ–≤–∫–æ–≤: –æ—Ç `H1` (—Å–∞–º–æ–≥–æ –±–æ–ª—å—à–æ–≥–æ) –¥–æ `H6` (—Å–∞–º–æ–≥–æ –º–∞–ª–µ–Ω—å–∫–æ–≥–æ). –î–ª—è –∏—Ö –≤—ã–¥–µ–ª–µ–Ω–∏—è –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Ä–µ—à—ë—Ç–∫–∏ `#`, –ø—Ä–∏ —ç—Ç–æ–º –µ—Å—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ç–æ–Ω–∫–æ—Å—Ç–µ–π:
    
+ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–µ—à—ë—Ç–æ–∫ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —É—Ä–æ–≤–Ω—é –∑–∞–≥–æ–ª–æ–≤–∫–∞: –æ–¥–Ω–∞ –¥–ª—è –ø–µ—Ä–≤–æ–≥–æ —É—Ä–æ–≤–Ω—è, –¥–≤–µ –¥–ª—è –≤—Ç–æ—Ä–æ–≥–æ –∏ —Ç–∞–∫ –¥–∞–ª–µ–µ;

+ –º–æ–∂–Ω–æ —Å—Ç–∞–≤–∏—Ç—å –ª—é–±–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∑–Ω–∞–∫–æ–≤, –∏ –Ω–∞ —Ç–∏–ø –∑–∞–≥–æ–ª–æ–≤–∫–∞ —ç—Ç–æ –Ω–µ –ø–æ–≤–ª–∏—è–µ—Ç;
+ –º–µ–∂–¥—É –∑–∞–≥–æ–ª–æ–≤–∫–æ–º –∏ –∑–Ω–∞–∫–∞–º–∏ –Ω–µ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –ø—É—Å—Ç—ã—Ö —Å—Ç—Ä–æ–∫.
